# 食用说明

## 食用方法：

1. 本库所有函数均采用“类型名+功能名”方式命名。
2. 本库所有函数均有中文注释，参照注释内容传递参数与调用。
3. 欢迎大家对函数进行补充，QQ群：1020853347

## 导入方法

### 第一种：

直接go get github.com/lh88888/bigtires

或

在使用mod管理自动导入

### 第二种

下载本项目源码文件夹，放进你所在的项目中即可使用
